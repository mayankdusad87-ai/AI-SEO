# AI Real Estate SEO Generator

AI-powered SEO keyword and hashtag generator for real estate projects.

Built using:
- Streamlit
- Grok API
- Python

---

# Features

- SEO Keyword Generation
- Long-tail Keyword Suggestions
- Instagram Hashtag Generation
- Real Estate Specific AI Prompting

---

# Tech Stack

- Streamlit
- Python
- xAI API
- OpenAI SDK

---

# Installation

## Clone Repository

```bash
git clone https://github.com/your-username/seo-generator.git
```

---

## Move Into Project

```bash
cd seo-generator
```

---

## Install Requirements

```bash
pip install -r requirements.txt
```

---

## Add Environment Variables

Create a `.env` file:

```env
XAI_API_KEY=your_api_key_here
```

---

# Run Application

```bash
streamlit run app.py
```
